Object & Scope

NEW: x-data, x-text

In this example we define some JavaScript objects. ("Variables")
In Alpine.js those objects are connected to HTML Elements.
We will connect one object to the body-tag, and another one to div1.

Output in the body

Steps in this lecture

  1. Define x-data="{bodyObj: 'body'}" in body tag.
  2. Output of "bodyObj" with x-text="bodyObj"
  3. Defining Div-1-Object: x-data="{div1obj: 'div1'}"
  4. Output of Div-1-Object in Div1
  5. Output of "bodyObj" in Div2
  6. Showing the cases that will NOT work!

Things to remember:

  • As well as the definition, as the output of Objects are bound to HTML-Elements in Alpine.js
  • If a variable is not set you can use a placeholder.
  • Variables in JS - Objects of course can be strings, numbers, booleans but also other objects! (see next lecture)

Important Constraints!

  • Objects can only be used in the same HTML-Element or nested HTML-Elements
  • A HTML-Element can only contain or output one specific object.

What will NOT work

  • NOT: Output of an object, in Element where other object is defined.
  • NOT: Output of an object, in a parent element or sibling element.

In Doubt, always use the console in Chrome DevTools to see if there is an error !
