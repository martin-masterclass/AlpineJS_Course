Steps

  1. Create a folder js and in this folder a file components.js.
  2. Outsource the function myComponent into the file.
  3. Include the outsourced file via a script tag.
  4. Copy example 2 into example 4.
  5. In component.js create another function myComponent2 with the name property and a copy of the "hello" function.
  6. Slightly change the Hello function in this copied function.
  7. In the new example 4 use the new component: myComponent and check if everything works.
